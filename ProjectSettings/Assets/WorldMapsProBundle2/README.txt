Thank you for purchasing!

World Maps Pro Bundle 2 is our most complete and professional suite of map assets for Unity, including:

- World Political Map Globe Edition 2
- World Political Map 2D Edition 2
- World Map Strategy Kit 2
- World Maps and Weather Symbols

Import instructions
-------------------

As a customer of this bundle, you can download any of these assets using the linkgs below:
- World Map Globe Edition 2: https://assetstore.unity.com/packages/tools/gui/world-map-globe-edition-2-150643
- World Map 2D Edition 2: https://assetstore.unity.com/packages/tools/gui/world-map-2d-edition-2-151238
- World Map Strategy Kit 2: https://assetstore.unity.com/packages/templates/systems/world-map-strategy-kit-2-150938
- World Flags & Weather Symbols Pack: https://assetstore.unity.com/packages/2d/textures-materials/world-flags-and-weather-symbols-69010

Please note that the packages World Political Map 2D Edition and World Map Strategy Kit CAN NOT be imported in the same project since they are meant to be exclusive.
However, you can combine any of them with World Political Map Globe Edition in the same project (as well as the World Maps and Weather Symbols pack of course).
